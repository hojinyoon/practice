# Project Plan

## Goal
Create a reproducible data analysis using Quarto that explores airport traffic, Monte Carlo integration, and calcium data.

## Needs
- Tidy datasets
- Clear visualizations
- Reproducible workflow
- Organized repository

## Steps
1. Build the Quarto document with all required sections.
2. Make sure the document renders to PDF successfully.
3. Add all needed files to the repository.
4. Create a README describing the project and repo contents.
5. Create a plan document for the project and repo maintenance.
6. Use a dev branch for edits.
7. Open and merge at least one pull request into main.
8. Create and close at least two issues.
9. Make multiple meaningful commits with clear messages.
