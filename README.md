# Reproducible Data Analysis (HW #4.3)

## Overview
This project is a reproducible data analysis created using Quarto. It includes an analysis of the busiest airports, Monte Carlo numerical integration, and a data wrangling task using a calcium dataset.

## Data Sources
- Calcium dataset (provided in assignment)
- Airport data manually constructed from assignment context

## Project Plan
The project focuses on:
- Creating clean visualizations
- Using tidy data principles
- Applying Monte Carlo simulation
- Comparing plan-informed vs generic GenAI outputs

## Repository Structure
- 'Reproducible-Data-Analysis.qmd'; main analysis file
- 'Reproducible-Data-Analysis.pdf'; rendering output
- 'calcium.csv'; dataset
- 'plan.md'; planning document
